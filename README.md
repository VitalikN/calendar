# Calendar

## The Calendar project is a feature-rich web application designed for efficient task management within a calendar interface. It includes essential functionalities such as task creation and editing, task filtering, label customization, and display of worldwide holidays. The project offers an intuitive interface, responsive design, and extensive customization options for users.

## Technologies Used

- **[Next.js](https://nextjs.org/)**: A React framework for production-grade applications with server-side rendering and static site generation.
- **[Redux Toolkit](https://redux-toolkit.js.org/)**: Used for efficient state management and asynchronous logic.
- **[Formik](https://formik.org/)**: For building scalable, performant forms with ease in React.
- **[emotion/styled](https://emotion.sh/docs/styled)**: A library for creating styled components in React, which allows you to describe styles directly in the code using a special syntax.
- **[Yup](https://github.com/jquense/yup)**: Used for object schema validation.
- **[React Icons](https://react-icons.github.io/react-icons/)**: Popular icons in your React projects.
